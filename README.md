# 담당 로직

## User API

- 아이디 중복확인
- 이메일 중복확인
- 회원가입
- 이메일 인증
- 아이디 찾기
- 비밀번호 찾기
- 비밀번호 재설정
- 로그인 (Mobile Environment)
- 로그인 (Web Environment)
- 로그아웃 (Web Environment)
- 토큰 재발급
- 휴면 계정 처리
- 회원탈퇴
- 내 정보
- 이용제한 내역 조회
- 블랙리스트 내역 조회
- 강의평가 신고하기
- 시험정보 신고하기


---

## Admin API

-  관리자 로그인
-  신고된 게시글 리스트 불러오기
-  신고된 게시글 자세히 보기(강의평가)
-  신고된 게시글 자세히 보기(시험정보)
-  강의평가 삭제 및 유저 정지 조치
-  시험정보 삭제 및 유저 정지 조치
-  강의평가 삭제 및 유저 블랙리스트 조치
-  시험정보 삭제 및 유저 블랙리스트 조치
-  신고된 시험정보 이상 없음 조치
-  신고된 강의평가 이상 없음 조치
